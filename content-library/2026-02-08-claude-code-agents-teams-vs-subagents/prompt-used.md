# Prompt Used for Infographic Generation

**Tool:** Nano Banana MCP (generate_infographic)
**Model:** gemini-2.0-flash-exp-image-generation
**Date:** 2026-02-08

## Infographic Prompt

```
You are a visual infographic designer.
Generate a single hand-drawn infographic image.

TOPIC:
Sub-Agents vs Agent Teams

CONTENT TO VISUALIZE:

Section 1 - SUB-AGENTS (orange heading):
   • Focused contractors
   • Spin up, do one task, report back
   • Fast and cheap (~1x tokens)
   • Can't talk to each other
[gear icon doodle]

Section 2 - AGENT TEAMS (orange heading):
   • Work like a real team
   • Share context, coordinate
   • Higher cost (~7x tokens)
   • Collaborative output
[team/people icon doodle]

Section 3 - WHEN TO USE (orange heading):
   • Sub-agents: parallel isolated tasks
   • Teams: work that builds on itself
   • Match tool to problem shape
[lightbulb icon doodle]

KEY TAKEAWAY:
"Function call? Sub-agent. Team meeting? Agent team."

STYLE:
- Hand-drawn on whiteboard/notebook
- Black + Orange (#E8611A) markers only
- Handwritten text, wobbly lines
- 1080x1350 portrait format

BOTTOM CTA:
"Follow @DiegoVences for more AI + Sales content"
```
