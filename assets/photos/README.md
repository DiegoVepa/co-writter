# Photos Directory

**Purpose:** Store Diego's professional photos for visual asset creation.

---

## File Structure

```
/assets/photos/
├── diego-cutout.png      # Transparent background, professional pose
├── diego-dark.png        # Dark/natural background version
├── generated/            # AI-generated composites via Nano Banana
│   └── [timestamp]-[description].png
└── README.md
```

---

## Photo Requirements

### diego-cutout.png
- **Background:** Transparent (PNG with alpha)
- **Pose:** Professional, approachable (standing, holding tablet or gesturing)
- **Attire:** Black polo or smart casual
- **Usage:** Light carousel slides, title slides
- **Recommended size:** 1000px+ height, maintain aspect ratio

### diego-dark.png
- **Background:** Natural or dark setting
- **Pose:** Professional, confident
- **Attire:** Casual/smart casual
- **Usage:** Dark carousel slides, closing slides
- **Recommended size:** 1000px+ height, maintain aspect ratio

### generated/
- **AI-generated composites** via Nano Banana MCP
- **Naming:** `YYYY-MM-DD-description.png`
- **Save prompt used** in accompanying `.md` file

---

## Adding Photos

1. Export photo with transparent background (for cutout)
2. Save as PNG format
3. Name exactly as specified above
4. Test in carousel templates to verify quality
