# Visual Brand Kit - Diego Vences Content

**Version:** 1.0
**Last Updated:** 2025-01-25

## Purpose

Define the visual identity for all content assets. This ensures consistency whether you're creating graphics yourself (Canva) or providing direction to a branding agency.

---

## Color Palette

### Primary Colors
| Color | Hex | Use |
|-------|-----|-----|
| **Orange** | #E67E22 | Headlines, CTAs, emphasis, @handle, borders |
| **Dark Charcoal** | #2D2D2D | Dark backgrounds, body text |
| **Cream** | #F5F0E6 | Light backgrounds |

### Secondary Colors
| Color | Hex | Use |
|-------|-----|-----|
| **White** | #FFFFFF | Text on dark backgrounds, card backgrounds |
| **Black** | #1A1A1A | Body text on light backgrounds |

### Brand Elements
- **Sparkles**: + + + in orange (#E67E22) as decorative accents
- **Handle**: @DiegoVences in orange pill or text at bottom of slides
- **Borders**: Orange rounded borders on cards and buttons

---

## Typography

### Headlines
- **Font:** Inter Bold or Montserrat Bold
- **Size:** 32-48px for carousel titles
- **Style:** Clean, no decorative fonts

### Body Text
- **Font:** Inter Regular or Open Sans
- **Size:** 16-20px
- **Line Height:** 1.5

### Callouts/Stats
- **Font:** Inter Bold
- **Size:** 24-36px
- **Use:** ROI numbers, before/after metrics

---

## Visual Style Principles

### DO:
- Clean, minimal layouts
- Lots of white space
- One main idea per slide/image
- Real screenshots of tools (n8n, Claude, etc.)
- Simple icons (outline style, not filled)
- Before/After comparisons with clear visual separation
- Hand-drawn style arrows or annotations (authentic feel)

### DON'T:
- Overly designed/polished (feels corporate)
- Stock photos of people
- Gradient overuse
- 3D illustrations
- Complex infographics
- AI-generated imagery (ironic for AI content)

## Carousel Slide Templates

### Slide 1: Title/Hook
```
┌─────────────────────────────┐
│                             │
│   [BOLD HOOK STATEMENT]     │
│                             │
│   Smaller context line      │
│                             │
│          [Icon]             │
│                             │
└─────────────────────────────┘
```
- Background: Deep Coral or Dark Charcoal
- Text: White
- One simple icon centered bottom

### Slide 2-5: Content Slides
```
┌─────────────────────────────┐
│  [Number or Icon]           │
│                             │
│  Headline Point             │
│                             │
│  • Supporting detail        │
│  • Supporting detail        │
│                             │
└─────────────────────────────┘
```
- Background: White or Soft Cream
- Headline: Deep Coral
- Body: Dark Charcoal

### Slide: Before/After
```
┌─────────────────────────────┐
│  BEFORE        │  AFTER     │
│  ───────       │  ───────   │
│  2 days        │  20 min    │
│  Manual        │  Automated │
│  $2K loss      │  $2K saved │
│                │            │
└─────────────────────────────┘
```
- Split layout
- Before: Red/coral tint
- After: Green tint or clean white

### Slide: Final CTA
```
┌─────────────────────────────┐
│                             │
│   Key Takeaway Line         │
│                             │
│   ─────────────────         │
│                             │
│   [Engagement Question]     │
│                             │
└─────────────────────────────┘
```
- Background: Dark Charcoal
- Text: White
- Question in Warm Orange

## Image Types by Content

### "What I Built" Content
- Screenshot of workflow (n8n, Make, etc.)
- Terminal/code snippet (clean, syntax highlighted)
- Before/after process diagram
- Simple flow: Input → Process → Output

### "Boring Business" Content
- Simple ROI chart (bar or single number highlight)
- Text message mockup (appointment reminder)
- Before/after metrics comparison
- Icon representing industry (truck, tools, etc.)

### Thought Leadership
- Single powerful stat on plain background
- Quote card (your insight)
- Simple 2x2 matrix or framework diagram

## Icon Style

### Source
- Use Lucide icons (https://lucide.dev) or similar outline style
- Consistent stroke width (2px)
- Single color per icon

## Canva Template Recommendations

Create these 5 base templates:

1. **Carousel Title Slide** - Hook + icon
2. **Carousel Content Slide** - Number + headline + bullets
3. **Before/After Comparison** - Split layout
4. **Stat Highlight** - Single big number + context
5. **Quote/Insight Card** - Your words, simple background

---

## Quality Checklist

Before publishing any visual:

- [ ] Uses brand colors only
- [ ] Text is readable (contrast check)
- [ ] One main idea per image
- [ ] Clean white space
- [ ] No stock photos
- [ ] Consistent with other recent posts
- [ ] Mobile-readable (test at small size)
